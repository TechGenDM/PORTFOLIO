## 🌌 About This Masterpiece

Welcome to my digital universe! This isn't just a portfolio; it's a highly over-engineered, beautifully animated, and aesthetically pleasing manifestation of my coding journey. I built this to showcase my projects, flex some animations, and drop a few memes along the way.

### ✨ What's under the hood?

- **⚡ Blazing Fast:** Powered by Next.js App Router.
- **🎨 Eye-Candy UI:** Tailwind CSS because writing pure CSS is so 2010.
- **📱 Responsive AF:** Looks gorgeous on your toaster, fridge, and occasionally a smartphone.
- **🐙 GitHub Integration:** Flexing public repos automatically.

---

## 💻 Getting Started (If you dare)

Want to run this locally and see the magic happen on `localhost:3000`?

```bash
# 1. Clone this bad boy
git clone https://github.com/TechGenDM/PORTFOLIO.git

# 2. Enter the matrix
cd PORTFOLIO

# 3. Install the black magic (dependencies)
npm install

# 4. Fire it up!
npm run dev
```
